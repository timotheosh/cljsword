* cljsword

Using Crosswire's Open Source Bible software library, JSword with Clojure.

** Installation

git clone from https://github.com/timotheosh/cljsword.git
Build with lein
#+BEGIN_SRC sh
cd cljsword/ && lein uberjar
#+END_SRC
